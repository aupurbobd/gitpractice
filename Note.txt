Working Directory --> Stagging area      --> Local Folder/Repository --> Remote Repository
/gitpractice      --> git add "filename" --> git commit              --> git push
git initialiaztion == git init
check status = git status 

add all files from the working Directory to the stagging area: git add .

moving files to the local repo: git commit Note.txt -m "moving the first file to the local repo."

 to see the log in one line: git log --oneline

 bring back the file from local repo to stagging area: git checout
